🚀 БЫСТРЫЙ СТАРТ:

   Windows:
   ────────
   1. Откройте START_HERE.md
   2. Настройте .env файлы (2 минуты)
   3. Запустите: start.bat
   4. В новом терминале: cd mobile && npx expo start

   Mac/Linux:
   ──────────
   1. Откройте START_HERE.md
   2. Настройте .env файлы (2 минуты)
   3. Запустите: ./start.sh
   4. В новом терминале: cd mobile && npx expo start

🎯 ЧТО ОСТАЛОСЬ СДЕЛАТЬ ВАМ:

   ⬜ Настроить backend/.env (2 минуты)
      - SUPABASE_URL
      - SUPABASE_KEY
      - SUPABASE_SERVICE_KEY

   ⬜ Настроить mobile/.env (1 минута)
      - API_URL=http://ВАШ_IP:8001/api

   ⬜ Запустить: start.sh или start.bat

   ⬜ Открыть mobile и запустить: npx expo start

   ⬜ Отсканировать QR код в Expo Go
